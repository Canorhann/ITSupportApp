# 🛠️ Corporate IT Support Ticket System

A robust, console-based Java application designed to simulate a real-world IT support helpdesk environment. This project demonstrates core Object-Oriented Programming (OOP) principles and dynamic data management.

## 🚀 Features
* **Dynamic Ticket Creation:** Users can generate new support tickets with unique IDs.
* **Status Tracking:** Tickets are created with an "Open" status and can be updated to "Resolved".
* **Live Registry:** View all active and past tickets in a clean, formatted list.
* **Interactive CLI:** Built with a continuous `while` loop and `Scanner` for seamless user interaction.

## 💻 Tech Stack
* **Language:** Java (JDK 17+)
* **Architecture:** Object-Oriented Programming (Classes, Constructors, Encapsulation)
* **Data Structures:** Java Collections (`ArrayList`)

## ⚙️ How to Run
1. Clone this repository to your local machine.
2. Compile the code using standard Java tools or open it in an IDE like VS Code/IntelliJ.
3. Run the `App.java` file to start the interactive terminal interface.
